## 3. 🛑 คำสั่งปิดระบบเพื่อเตรียมแก้ไข (Shutdown Protocol)

เมื่อนายท่านพร้อมจะลงมือผ่าตัด ให้ใช้คำสั่งนี้เพื่อ **"เคลียร์พื้นที่"** ให้สะอาดหมดจดค่ะ:

```bash
# 1. หยุดและลบ Container ทั้งหมด (รวมถึงตัวที่ชื่อเพี้ยนๆ)
docker rm -f $(docker ps -a -q)

# 2. ลบ Network ที่ค้างอยู่ (เพื่อความชัวร์)
docker network prune -f

# 3. (Optional) ลบ Image เก่าทิ้งด้วย (ถ้าจะ Build ใหม่แบบ Clean 100%)
# docker rmi acatest01/mcp-core:latest acatest01/mozart-rag:latest
```

**คำเตือน:** คำสั่ง `docker rm -f $(docker ps -a -q)` คือการ **"ล้างบาง"** Container ทุกตัวในเครื่องนะคะ (ถ้านายท่านมีโปรเจกต์อื่นรันอยู่ด้วย ให้ระวังค่ะ!)

## 4. 📝 ตัวอย่าง Dockerfile ฉบับปรับปรุง (Refactored Version)

Architecta ขอร่าง **"Dockerfile ฉบับปรับปรุง"** สำหรับ `mcp_core_v2` ให้นายท่านดูเป็นแนวทางค่ะ:

```dockerfile
# Stage 1: Builder (โรงงานผลิต)
FROM python:3.12-slim-bookworm AS builder

# ลงเครื่องมือที่จำเป็นจริงๆ (ตัด libpq-dev ออกถ้าไม่ใช้ DB)
RUN apt-get update && apt-get install -y --no-install-recommends \
    gcc \
    g++ \
    && rm -rf /var/lib/apt/lists/*

# สร้าง venv และลง Library
RUN python -m venv /opt/venv
ENV PATH="/opt/venv/bin:$PATH"
COPY requirements.txt .
RUN pip install --no-cache-dir --upgrade pip && \
    pip install --no-cache-dir -r requirements.txt && \
    pip install --no-cache-dir fastapi uvicorn[standard]

# -----------------------------------------------------------

# Stage 2: Production (สินค้าพร้อมขาย)
FROM python:3.12-slim-bookworm

# ลง curl ไว้เช็คสุขภาพ (เบากว่า python script)
RUN apt-get update && apt-get install -y --no-install-recommends curl && rm -rf /var/lib/apt/lists/*

# สร้าง User ธรรมดา (เพื่อความปลอดภัย)
RUN useradd -m -u 1000 appuser

# ตั้งค่า Environment
ENV PYTHONUNBUFFERED=1 \
    PYTHONDONTWRITEBYTECODE=1 \
    PYTHONPATH=/app \
    PATH="/opt/venv/bin:$PATH"

WORKDIR /app

# ก๊อปปี้ venv จากโรงงาน
COPY --from=builder /opt/venv /opt/venv

# ก๊อปปี้โค้ด (เจ้าของไฟล์ต้องเป็น appuser)
COPY --chown=appuser:appuser . .

# สร้างโฟลเดอร์ Log (ต้องให้สิทธิ์ appuser เขียนได้)
RUN mkdir -p logs output && chown -R appuser:appuser logs output

# เปลี่ยนร่างเป็น User ธรรมดา
USER appuser

EXPOSE 5001

# หมอคนใหม่ (ใช้ curl)
HEALTHCHECK --interval=30s --timeout=5s --start-period=5s --retries=3 \
    CMD curl -f http://localhost:5001/health || exit 1

CMD ["uvicorn", "api:app", "--host", "0.0.0.0", "--port", "5001"]
```
