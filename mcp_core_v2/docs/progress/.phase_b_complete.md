## 📋 **Design Decisions**

### **1. Return Type: Dict (Not List)**
**Locked per plan:**
```python
# ✅ CORRECT
def place_all_devices() -> Dict[str, Any]:
    return {
        'outlets': [...],
        'lights': [...],
        'switches': [...],
        'validation': {...}
    }

# ❌ WRONG (never do this)
def place_all_devices() -> List:
    return [outlet1, outlet2, ...]
```

### **2. Circuit Assignment: Deferred to MCP**
**Locked per plan:**
```python
# Phase B places devices
device = {
    'id': 'OUT-001',
    'position': (1000, 0),
    'circuit': None  # ← Not assigned yet
}

# Phase D (future) assigns from MCP
device['circuit'] = mcp_result.circuits[0].id
```
