# ตารางข้อมูลสำคัญ (Key Tables)

เอกสารนี้รวบรวมตารางสำคัญที่ใช้บ่อยในการออกแบบระบบไฟฟ้า เพื่อให้ค้นหาได้ง่ายและรวดเร็ว

### สรุปพิกัดกระแสสาย THW ในท่อร้อยสาย (Quick Reference)
เพื่อให้ง่ายต่อการค้นหาและอ้างอิง:
- สาย THW ขนาด **1.5 ตร.มม.** เดินในท่อร้อยสาย มีพิกัดกระแส **16 A**
- สาย THW ขนาด **2.5 ตร.มม.** เดินในท่อร้อยสาย มีพิกัดกระแส **24 A**
- สาย THW ขนาด **4.0 ตร.มม.** เดินในท่อร้อยสาย มีพิกัดกระแส **32 A**
- สาย THW ขนาด **6.0 ตร.มม.** เดินในท่อร้อยสาย มีพิกัดกระแส **41 A**
- สาย THW ขนาด **10 ตร.มม.** เดินในท่อร้อยสาย มีพิกัดกระแส **57 A**
- สาย THW ขนาด **16 ตร.มม.** เดินในท่อร้อยสาย มีพิกัดกระแส **76 A**
- สาย THW ขนาด **25 ตร.มม.** เดินในท่อร้อยสาย มีพิกัดกระแส **101 A**
- สาย THW ขนาด **35 ตร.มม.** เดินในท่อร้อยสาย มีพิกัดกระแส **125 A**
- สาย THW ขนาด **50 ตร.มม.** เดินในท่อร้อยสาย มีพิกัดกระแส **151 A**

## 2. ตัวคูณปรับค่า (Derating Factor)
**อ้างอิง:** วสท. 2564 ตารางที่ 5-8 (กลุ่มวงจรย่อย หรือสายเคเบิลหลายแกน)

| จำนวนตัวนำที่มีกระแสไหล (เส้น) | ตัวคูณปรับค่า (Cg) |
| :---: | :---: |
| 1-3 | 1.00 |
| 4-6 | 0.80 |
| 7-9 | 0.70 |
| 10-20 | 0.50 |
| 21-30 | 0.45 |
| 31-40 | 0.40 |
| 41 ขึ้นไป | 0.35 |

**ตัวอย่างการใช้งาน:**
- ถ้าเดินสายไฟ 6 เส้นในท่อเดียวกัน ให้ใช้ตัวคูณ 0.80
- พิกัดกระแสใหม่ = พิกัดกระแสเดิม x 0.80

## 3. ข้อกำหนดเครื่องตัดไฟรั่ว (RCD)
**อ้างอิง:** วสท. 2564 ข้อกำหนดการติดตั้ง

- **ห้องน้ำ:** วงจรเต้ารับและเครื่องทำน้ำอุ่นในห้องน้ำ ต้องติดตั้ง RCD/RCBO ที่มีพิกัดกระแสรั่วไหลไม่เกิน **30 mA** และตัดวงจรภายใน **0.04 วินาที**
- **พื้นที่อันตราย:** พื้นที่ที่มีความชื้นสูง หรือจุดที่ผู้ใช้สัมผัสเครื่องใช้ไฟฟ้าได้ง่าย ต้องติดตั้ง RCD
